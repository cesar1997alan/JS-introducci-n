# Introducción JS

Repositorio de práctica personal para reforzar fundamentos de **JavaScript** 

##  Objetivo
Consolidar bases de JavaScript mediante ejercicios cortos por tema:
- Variables, tipos y operadores
- Condicionales y loops
- Arreglos y métodos (`forEach`, `map`, etc.)
- Objetos, prototypes y POO
- Clases (ES6)
- Promesas y **Fetch API**
- Lectura de datos desde archivo JSON (`empleados.json`)
